# Practice-server
쇼핑몰을 주제로 CRUD, Spring Security 연습하는 Repository

# 📋 Commit Message Convention (이모지 + 소문자 적용)
| Tag           | Description             |
|---------------|-------------------------|
| `✨ feat`      | 새로운 기능 추가        |
| `🐛 fix`      | 버그 수정              |
| `📝 docs`     | 문서 추가, 수정, 삭제  |
| `♻️ refactor` | 코드 리팩토링         |
| `🚀 chore`    | 기타 변경사항          |
